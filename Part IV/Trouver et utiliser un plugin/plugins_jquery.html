Trouver et utiliser un plugin

La biblio jQuery a été écrite de telle manière qu'il est très simple de l'étendre en installant des modules additionnels, connus sous le nom d'extensions ou de plugins. De nombreux sites Web se sont spécialisés dans les plugins jQuery.

II./ Quelques exemples de plugins

1.) Parseur RSS / Atom De nombreux sites Web proposent des flux de données au format RSS et/ou Atom. En utilisant un parseur, je peux récupérer ces flux de données et les intégrer à mon site. Justement, "FeedEk" excelle dans ce domaine. Je me rends sur la page de téléchargement du plugin et cliquer sur Donwload. Je vais maintenant parser un des flux RSS proposés par le site lemonde.fr. Je me rends sur la page suivante : www.lemonde.fr/rss/ ...et je choisis un des flux proposés, en cliquant du bouton droit sur l'icône XML correspondante et je sélectionne Copier le raccourci dans le menu contextuel. Réf Exemple 2 : FeedEk 2.) Validation de formulaires Le plugin "Validate" est une formidable solution pour valider facilement un formulaire. Pour utiliser le plugin "Validate", il suffit d'exécuter la méthode validate() lorsque le DOM est disponible : $(document).ready(function() { $('#idForm').validate(); }); Pour définir le type de validation souhaité sur un champ du formulaire, je lui affecte la classe correspondante : - required : champs requis - email : adresse e-mail - url : adresse URL - date : date - number : nombre - creditcard : numéro de carte de crédit Réf Exemple 3 : Validate Et, pour faire afficher les messages non pas en anglais, mais en français par exemple, j'inclus le script qui se trouve dans : dist / localization / ==> messages_fr.min.js 3.) Un menu déroulant à un ou plusieurs niveaux Il est souvent nécessaire de mettre en place un menu déroulant dans un site. Il existe un plugin permettant de le faire très facilement : Simple jQuery Dropdowns Le plugin utilisé est le jquery.dropdownPlain.js. Le simple fait d'utiliser ce script me dédouane de tout autre code jQuery. Pour construire mon menu, il me suffira de définir une liste < ul>< li>< /li>< /ul> à un ou plusieurs niveaux. Réf Exemple 4 : dropdownPlain Si je dois mettre en place sur un de mes sites Web, ce plugin devrait me faciliter grandement la tâche. Je dois penser à insérer les fichiers suivants : - js/jquery.dropdownPlain.js - css/style.css 4.) Cartographie Pour afficher sur mon site Web une carte géographique centrée sur un point particulier, j'utilise le plugin "gMap" qui interface le système de cartographie Google Maps. Je me rends sur cette page et je télécharge la dernière version du plugin. Je crée un nouveau document HTML et j'y insère une balise < div> dans le corps du document. Je donne à cette balise la dimension que je veux lui voir occuper dans la page et j'affecte la valeur hidden à son attribut overflow : <div id="map1" style="width: 800px; height: 600px; border: 1px solid #777; overflow: hidden;"> </div> Pour accéder à toute la puissance de Google Maps, j'utilise trois balises < script> pour faire référence à : - la bibliothèque jQuery - à l'API de Google Maps - au plugin jquery.gmap.min.js Pour afficher la carte du monde dans la balise < div id="map1">, il me suffit de lui appliquer la méthode gMap() dès que le DOM est disponible : $(function() { $('#map1').gMap(); }); Plusieurs paramètres peuvent être passés à la méthode gMap(). Entre autres : - La latitude et la longitude : propriétés "latitude" et "longitude" - address : des informations sur le centre d'affichage - zoom : le facteur de zoom (entre 1 et 19) - maptype : le type de carte (HYBRID, TERRAIN, SATELLITE, ROADMAP) Note : pour avoir la liste complète des paramètres qui peuvent être passés à la méthode gMap(), je peux consulter la documentation. Et pour connaître la latitude et la longitude d'une ville, je vais sur le site de Google Maps, je tape le nom de la ville dans le champ de recherche et j'apppuie sur la touche Entrée de mon clavier. La carte se centre sur la ville avec une icône. Je fais un clic droit sur cette icône et je choisis Plus d'infos sur cet endroit. Le champ de recherche est alors mis automatiquement à jour avec la latitude et la longitude. Pour termine, voici un exemple concret d'utilisation. Je vais afficher une carte centrée sur la ville d'Albi, située aux coordonnées (43.92, 2.14). Le texte "Ici Albi" sera affiché dans une bulle, le facteur de zoom sera égal à 10 et la carte sera de type "vue en relief". Voici le code à utiliser : $('#map').gMap({ markers: [{ latitude: 43.92, longitude: 2.14, html: "Ici Albi", popup: true }], maptype: 'TERRAIN', zoom: 10 }); Réf Exemple 5 : gMap Remarque : pour la nouvelle version de Google Maps, il faut obtenir une clé apparemment, pour utiliser leur carte. Remarque2 : il semblerait que ce plugin soit obsolète. Je vais donc plutôt utiliser l'Api Google Maps directement.
